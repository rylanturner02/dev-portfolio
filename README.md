# rylanturner.dev

I refreshed this website from it's long-abandoned life as a Remix app, briefly in Vue.js, and finally in Svelte 5, which seemed the most enjoyable process to me.
The homepage is inspired by [Aerial from HTML5 UP](https://html5up.net/aerial), of which I wanted to recreate using a framework I'm comfortable with.

## Features
- Tileable [background image](https://www.freepik.com/free-vector/vector-seamless-forest-illustration-with-mountains-background-horizontally-repeatable_132100058.htm)
- Glossy overlay effect
- Text animations
- Preloading and state management for smoother experience

## Technology
- SvelteKit
- TypeScript
- Vercel
